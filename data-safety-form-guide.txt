        Required: Yes
        Shared with third parties: No
        Ephemeral: No

        Purpose: For profile pictures only
        Required: Optional
        Shared with third parties: No
        Ephemeral: No

      - Precise location: NOT COLLECTED
      
   C. FINANCIAL INFO
      - NOT COLLECTED
      
   D. MESSAGES

      - SMS or MMS: NOT COLLECTED

      - NOT COLLECTED
      
   H. CALENDAR
      - NOT COLLECTED

   J. APP ACTIVITY

        Purpose: To improve user experience

        Ephemeral: No
      
      - In-app search history: NOT COLLECTED
      
   K. WEB BROWSING
      - NOT COLLECTED

        Purpose: App improvement and bug fixing

        Purpose: App improvement and bug fixing

2. DATA SECURITY PRACTICES:

   - Data is encrypted in transit: YES
   - Data can be deleted by request: YES
     Method: Users can contact support@akinyemi.ca to request data deletion
   - Follows Security practices:
     - Data is encrypted at rest
     - App has a mechanism to allow users to request their data be deleted
     - Limited data retention period, data is only retained as long as necessary
   - Committed to following the Play Families Policy: YES

3. DATA SHARING:

   - Data is not shared with third parties: CORRECT
   - Data is processed for legal purposes: YES
     (legal obligations, fraud prevention, security)

DATA DELETION MECHANISM:
Users can request deletion of their data by:
1. Emailing support@akinyemi.ca
2. Using the account deletion option in the app settings
3. Data will be deleted within 30 days of the request
